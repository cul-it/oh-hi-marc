# Oh Hi MARC!

A simple, browser-based MARC record viewer and diff tool.

## Features

- View MARC21 binary files (.mrc, .marc)
- Compare two MARC files side-by-side
- Color-coded field tags by category
- Toggle to hide unchanged fields in diff view
- All processing happens locally in your browser

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
